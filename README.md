# Property Listings API

A RESTful API built with ASP.NET Core and SQL Server, demonstrating modern .NET development practices.

## Features
- Full CRUD operations for property listings
- Built with ASP.NET Core 10
- Swagger UI for API documentation and testing
- Entity Framework Core with SQL Server (in progress)
- JWT Authentication (coming)
- xUnit Tests (coming)
- Azure deployment (coming)

## Tech Stack
- .NET 10 / ASP.NET Core
- Entity Framework Core
- SQL Server
- Swagger

## Running Locally
1. Clone the repo
2. Open in Visual Studio
3. Hit F5
4. Navigate to https://localhost:7183/swagger
